Audio Reader

Reads selected text aloud with Kokoro — in your browser (WebGPU/WASM) or via a local TTS server. Nothing leaves your machine.

In-browser engines run Kokoro locally with no server. The first browser run downloads the model (~300 MB for WebGPU, ~86 MB for WASM), cached by the browser afterwards.
Base URL of the server. The extension calls /v1/audio/speech and /v1/audio/voices under it. Point this at any OpenAI-compatible TTS server (e.g. a Chatterbox instance) to switch engines.
